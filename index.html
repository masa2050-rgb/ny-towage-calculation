Model Formulas & Assumptions

Docking

Formula: Base Cost = Docking Rate × "Docking" Line Items

Assumption: Base tug dispatch is determined by Length Overall (LOA). 2 tugs for <997', 3 tugs for ≥997', and 4 tugs for ≥1165'.

+40K NRT Fee

Formula: Size Cost = If NRT > 40,000, add $1,400 × "Docking" Line Items

Assumption: This is an additional flat premium added per docking line item when the Net Register Tonnage exceeds the threshold.

Fuel Surcharge

Formula: Fuel Cost = ⌈(Fuel Price - 2.00) / 0.10⌉ × $15 × Total Physical Tugs

Assumption: Calculated per physical tug using a $2.00 base. Every $0.10 increment (rounded up) adds a $15 surcharge.

Escort

Formula: Escort Cost = Escort Rate × Billed Hours (Actual Escort Time + Runtime) × Total Escorting Tugs

Assumption: Billed Hours is rounded up to the nearest 0.5hrs increment and is subject to a 2.0-hour minimum billing per tug.

Per harbor regulations, vessels 997 feet LOA and larger require mandatory tug escorts to safely transit restricted waterways like the Kill Van Kull (Zone 11), whereas smaller vessels do not.

"Escort Only" & Runtime-waiver

Formula: If Direction = Inbound, Escort+Docking Zone Time = Runtime ÷ 2. Escort Only Zone Time = Full Runtime.

Assumption: The outbound runtime leg is waived upon arrival ONLY for tugs that transition to docking. "Escort Only" tugs bill the full two-way transit runtime.
